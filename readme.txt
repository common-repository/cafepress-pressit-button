Add a Cafepress "Press It" Button to your posts to allow your readers to easily create Cafepress products from your images.

== Description ==

Add a Cafepress "Press It" Button to your posts and start earning affiliate revenue in minutes.

### Cafepress "Press It" Button Plugin Features ###

* Get money from Cafepress for each product that your readers order using your images.
* Let the reader click on any image and order it on a Cafepress product.
* Specify the button location, above or below each image.

== Installation ==

**Installing through the Wordpress Admin page:**

1. Search for "Cafepress Press It" in your WordPress Admin page.
1. Click the "Intall Now" link and then the "Activate" link.
1. Open the new menu item on the left hand labeled "Press It Button".

**Alternative installation methods:**

* Download this plugin, then upload through the WordPress admin (Plugins > Add New > Upload)
* Download this plugin, unzip the contents, then FTP upload to the `/wp-content/plugins/` directory

Note: If you overwrite the plugin using an FTP upload, you may lose some saved settings.
